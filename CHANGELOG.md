# Changelog

All notable changes to this project will be documented in this file.
Please, document here only changes visible to the client app.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added

- Initial release of @torqlab/generate-activity-image
- AI-powered image generation for activities using Pollinations.ai
- Automatic retry logic with configurable attempts
- Fallback support with default prompt mechanism
- Base64-encoded image data URL output
- Extensible provider pattern for future image generation services
- Comprehensive TypeScript type definitions
- Full JSDoc documentation for all functions and types
- GitHub Actions CI/CD workflows for testing, linting, and publishing
- ESLint and Prettier configuration for code quality
- Husky pre-push hooks for quality gates
